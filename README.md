# BookVault

A command-line library management application built with Python to practice object-oriented programming and fundamental software development concepts.

## About the Project

This project is a personal learning project designed to apply concepts learned while studying Python, particularly **classes and objects**.

The Library Manager will allow users to manage a collection of books and interact with them through a simple command-line interface.

## Planned Features

* Add books to the library
* Remove books from the library
* View the library collection
* Search for books
* Rate books (0 to 5 stars)
* Update books
 
* Store information about each book, such as:

  * Title
  * Author
  * Genre
  * Publication year
  * Availability status

Additional features may be added as the project develops.

## Learning Goals

This project is intended to provide practical experience with:

* Classes and objects
* Attributes and methods
* Object-oriented programming
* Lists of objects
* Managing collections of data
* Conditional statements and loops
* Error handling
* User input validation
* Code organization
* Git and GitHub
* Potentially storing data using JSON

## Project Status

**In development**

The project will be developed progressively as new Python concepts are learned.
The structure may change as the project develops.

## Purpose

The purpose of this project is to build a practical application while developing a stronger understanding of object-oriented programming and software development.

Rather than following a tutorial and copying an existing project, the application will be designed and developed progressively based on concepts learned during my Python journey.
